# CSXAML for Visual Studio

CSXAML adds Visual Studio authoring support for `.csxaml` files through the same shared tooling stack used by the language server.

The current extension packages:

- CSXAML document registration
- the shared language server
- completion, formatting, diagnostics, semantic tokens, and component navigation

This extension is built and versioned from the CSXAML repo release pipeline so the editor experience stays aligned with the shipped generator, runtime, and language spec.
