# Getting Started

Start here when adding CSXAML to an app or library for the first time.

Recommended order:

1. Review [prerequisites](prerequisites.md).
2. Follow the [quick start](quick-start.md).
3. Build the [Todo tutorial](../tutorials/todo-app.md).
4. Keep [build troubleshooting](../troubleshooting/build-and-generation.md) nearby for package and generator issues.

Normal app authors install the `Csxaml` package. The package brings the build targets, packaged generator payload, and runtime dependency needed by generated components.
