# Troubleshooting

Start with the area that matches the failure:

- [Build and Generation](build-and-generation.md)
- [Editor Extensions](editor-extensions.md)
- [Packages and Versions](packages-and-versions.md)
- [Runtime Behavior](runtime-behavior.md)

When reporting an issue, include:

- package version
- target framework
- relevant `.csxaml` source
- full build or editor diagnostic
- generated file path if the issue points into `obj`
- editor extension version, when editor behavior is involved
