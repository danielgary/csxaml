# CSXAML Hello World

This is the smallest generated-app sample. It starts from `App.csxaml`, creates
the startup window from CSXAML, and intentionally has no default WinUI XAML app
shell files.

It intentionally has no:

- `App.xaml`
- `App.xaml.cs`
- `MainWindow.xaml`
- `MainWindow.xaml.cs`

Build it from the repository root:

```powershell
dotnet build .\samples\Csxaml.HelloWorld\Csxaml.HelloWorld.csproj --no-restore
```
